2、数据分析在企业的日常经营分析中作用：现状分析、原因分析、预测分析

3、数据分析步骤：明确分析目的和思路=》数据收集=》数据处理=》数据分析=》数据展现=》报告撰写

4、分析框架体系化：以营销、管理理论作为指导，结合实际业务情况，搭建分析框架

5、常用的营销理论模型：4P、用户使用行为、STP理论、AWOT等

6、管理理论模型： PEST、5W2H、时间管理、生命周期、逻辑树、金字塔、SMART

7、数据来源：数据库、互联网、市场调查、公开出版物

8、数据处理：数据清洗、数据转化、数据提取、数据计算

9、数据挖掘是一种高级的数据分析方法，侧重解决分类、聚类、关联、预测

10、常用指标：

  • 平均数：将总体各单位的数量差异抽象化，代表总体的一般水平，掩盖总体内各单位的差异
  • 绝对数：总体在一定时间、地点的综合性指标，比如GDP、总人口
  • 相对数：两个有联系的指标对比计算得到的数值，反映对象之间数量联系程度的综合指标，比如总人数（绝对数）50人，相对数7成、6倍、1:4等。
  • 百分比、百分点：相对数中的一种
  • 频数：一组数据中个别数据重复出现的次数（绝对数），频率（相对数）每组类别次数与总次数的比值
  • 比例和比率
  • 倍数和番数（2的N次方倍）
  • 同比：与历史同期相比得到的数值。环比是与前一个统计期进行比较得到的数值，反映事物逐期发展的情况。

1、常用的数据分析方法论：

  • （行业分析）PEST分析法：对行业宏观环境的分析。一般包括政治、经济、技术和社会。
  • （广泛）5W2H：why（为何）、what（何事）、何人（who）、何时（when）、何地（where）、如何做（how）、何价（how much）
  • （业务问题专题分析）逻辑树：将问题的所有子问题分层罗列。遵循以下原则：
    • 要素化：相同问题归纳为要素。
    • 框架化：要素组织为框架，不重不漏
    • 关联化：各要素保持必要的相互关系 缺点：可能不全面
  • （整体经营情况分析）4P营销理论：产品、价格、促销、渠道
  • （用户行为）用户行为理论：认知、熟悉、试用、使用、忠诚

4、数据处理

1、数据处理步骤：

  • 数据清洗：将对于重复数据筛选删除，补充缺失数据，纠正错误数据
  • 数据加工：对数据进行信息提取、计算、分组、转换，形成要求的数据格式

2、处理缺失值的方法

  • 用一个样本统计量的值代替，通常为样本平均值
  • 用一个统计模型计算出的值代替，常用的模型有回归模型。判别模型
  • 删除
  • 暂时保留，在相应的分析中排除（数据量大，缺失的值不多）

5、数据分析方法、工具

1、数据分析法

数据分析作用 基本方法 数据分析方法
现状分析 对比 对比分析、平均分析、综合评价分析
原因分 细分 分组分析、结构分析、交叉分析、杜邦分析、漏斗图分析、矩阵关联分析、聚类分析
预测分析 预测 回归分析、时间序列、决策树、神经网络

2、对比分析法

  • 特点：可以非常直观地看出事物某方面的变化或差距，并可以准确、量化得表示出这种变化或差距
  • 分类：静态比较和动态比较
    • 静态比较：在同一时间条件下对不同总体指标的比较，横向比较
    • 动态比较：在同一总体条件下对不同时期指标数值的比较，纵向比较
  • 注意事项
    • 指标的范围，计算方法，计量单位一致
    • 对比的对象有可比性
    • 对比的指标类型要一致

3、分组分析法

  • 特点：根据数据分析对象的特征，按照一定的指标，把数据分析对象划分为不同的部分和类型进行研究，揭示内在的联系和规律
  • 关键：确定组数与组距
  • 等距分组步骤
    • 确定组数，根据数据的大小
    • 确定组间距，组距=（最大值-最小值）/组数
    • 划分
  • 不等距分组

4、结构分析法

  • 特点：分析总体内的各部分与总体之间进行对比（各部分占总体的比重）

5、平均分析法

  • 特点：运用计算平均数的方法来反映总体在一定时间、地点条件下某一数量特征的一般水平。
  • 作用：对比同类现象在不同条件之间的差异程度，比用总量指标更有说服力
  • 平均指标包括：算数平均数、调和平均数、几何平均数、众数和中位数

6、交叉分析法

  • 特点：通常用于分析两个变量之间的关系（两张二维表有相同的字段，合并在一起）

7、综合评价分析法

  • 特点：运用多个指标对多个参评单位进行评价的方法
  • 步骤：
    • 确定综合评价指标体系，即包含哪些指标
    • 收集数据，对不同计量单位的指标数据进行标准化处理
    • 确定指标体系中各指标的权重
    • 汇总计算综合评价指数
    • 根据数据进行排序，得出结论
  • 权重确定方法：专家访谈法、德尔菲法、层次分析法，主成分分析法、因子分析法、回归分析法
  • 目标优化矩阵：将纵轴上的项目依次与横轴上的项目对比，由专家投票表决，纵轴上的项目比横轴上的项目重要，那么可以在两个项目相交的格子中填1，否则填0，最后将每行数字进行相加，根据合计数据进行排序。
人才评价 人品 动手能力 创新意识 教育背景 合计 排序
人品 x 0 1 1 2 2
动手能力 1 x 1 1 3 1
创新意识 0 0 x 1 1 3
教育背景 0 0 0 x 0 4

8、杜邦分析法

  • 特点：利用各主要财务指标间的内在联系，对企业财务状况及经济效益进行综合分析评价的方法
  • 内容：将若干个用于评价企业经营效率和财务状况的比率按其内在联系有机地结合起来，形成一个完整的指标体系，并最终通过权益收益率来综合反映。
  • 优点：采用金字塔结构，⑩财务哔哩哔哩分析的层次更清晰、条理更突出，简洁明了地表达各财务指标之间的关系。

9、漏斗图分析法

  • 特点：业务流程分析，适合业务流程比较规范、周期长、流程复杂

10、矩阵关联分析法

  • 内容：根据事物的两个重要属性作为分析的依据，进行分类关联分析。属性A和属性B分别为x，y轴，根据属性的程度划分，分为4个象限，将要分析的事物投射到象限内。
  • 特点：解决资源分配

6、数据展现

1、通过关系选择图表

  • 类型：成分（饼图、柱形图、条形图）、排序（柱形图、条形图、气泡图）、时间序列（柱形图、折线图）、频率分布（柱形图、条形图、折线图）、相关性（柱形图、条形图、散点图、气泡图）、多重数据比较（雷达图）

2、制作图表步骤

  • 确定要表达的主题或目的
  • 确定图表类型
  • 制作图表
  • 检查是否真实有效地展示数据
  • 检查是否表达了观点

3、主次图

  • 按照发生频率的高低顺序绘制的直方图，可以用来分析质量问题中的主要因素

4、旋风图（对称条形图）

  • 同一事物在某个活动后的变化
  • 因果关系（例如价格和销量）
  • 两个类别之间不同指标的比较（例如某班男生和女生各项学科成绩对比）

7、图表专业化

1、图表信息

  • 包含标题、图例、横纵坐标数值的单位、脚注（特殊数据背后的原因）、资料来源（可选）

2、饼图注意事项

  • 按照时钟表盘的刻度，把数据从12点钟的位置开始排列，最重要的成分紧靠12点钟的位置（符合阅读习惯）
  • 数据项最好在5项以内，超过5项的做复合饼图
  • 不使用图例，直接标注
  • 尽量不使用标签连线、3D效果
  • 扇区使用白色边框线

3、柱形图

  • 同一数据序列使用相同的颜色
  • 不使用倾斜的标签
  • 横轴标签文字过长可以使用条形图代替
  • 纵坐标刻度一般从0开始
  • 最好添加数值（若有数值则可以去掉纵坐标刻度线和网格线）

4、条形图

  • 同一数据序列使用相同的颜色
  • 尽量让数据由大到小排序
  • 不使用倾斜标签
  • 最好添加数据标签

5、折线图

  • 折线图相对粗一些，比网格线、坐标轴更突出
  • 线条一般不超过5条
  • 不使用倾斜标签
  • 纵坐标轴一般刻度从0开始
  • 实际值用实线，预测值用虚线

8、数据分析报告

1、数据分析报告

  • 根据数据分析原理和方法，运用数据来反映、研究和分析某项事物的现状、问题、原因、本质和规律，并得出结论，提出解决办法

2、原则

  • 规范性（前后一致、术语规范）
  • 重要性（体现数据分析的重点）
  • 谨慎
  • 创新

3、作用

  • 展示分析结果
  • 验证分析质量
  • 提供决策参考

4、分类

  • 专题分析报告
  • 综合分析报告
  • 日常数据通报

5、专题分析报告

  • 是对某一方面或某一问题进行专门研究，为决策提供参考和依据
  • 单一性：不要求反映事物的全貌，针对某一问题
  • 深入性：对问题进行具体描述，对原因进行分析，并提出切实可行的解决办法

6、综合分析报告

  • 全面评价
  • 全面性
  • 联系性：相关联的现象综合分析。联系的重点是比例关系和平衡关系，分析他们是否协调

7、日常数据通报

  • 以定期数据分析报表为依据，翻译计划执行情况，并分析其影响和形成原因
  • 进度性：计划执行的进度与时间的进展结合
  • 规范性：
    • 反映计划执行的基本情况
    • 分析完成或未完成的原因
    • 总结计划执行中的成绩和经验，找出存在的问题
    • 提出措施和建议
  • 时效性

8、报告结构

  • 标题
    • 解释基本观点
    • 概括主要内容
    • 交代分析主题
    • 提出问题
  • 目录
    • 一、分析背景与目的
    • 二、分析思路
    • 三、分析正文
      • 1、产品分析
      • 2、渠道分析
      • 3、价格分析
      • 4、促销分析
      • 四、总结与建议
  • 前言
    • 此次分析的意义
    • 要解决什么问题，达到什么目的
    • 怎么分析，从什么方面入手
  • 正文
  • 结论
  • 建议
  • 附录
